Thanks for downloading RapidFeedPlus! This is a small program designed to repair Fusion360's exported CNC Gcode (.nc) because of the limits imposed on the free version, where rapid modes are slowed to the same speed as cutting speeds. This has only been tested when exported as GRBL currently.
